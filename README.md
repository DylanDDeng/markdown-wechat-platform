# WeChat Preview (Obsidian Plugin)

Preview the **currently open note** with WeChat-styled HTML (Swiss Klein / Klein Blue) and copy rich HTML into the WeChat editor.

## Dev

```bash
npm install
npm run dev
```

Build output is in `dist/` (`dist/main.js`, `dist/manifest.json`, `dist/styles.css`).

## Install (manual)

Copy `dist/` contents into:

`<Vault>/.obsidian/plugins/wechat-markdown-note/`

Then in Obsidian:

- Settings → Community plugins → enable `WeChat Preview`
- Command palette → run `Open WeChat Preview`
